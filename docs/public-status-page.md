# 公网状态页边界

`chinavpn.mikezhuang.cn` 推荐作为低权限公网入口，而不是完整 WireGuard 管理面板。

## 可以公网提供

- 登录保护后的服务说明。
- 只读状态，例如“服务在线”“维护中”“请先连接 VPN 后访问管理面板”。
- 客户端安装说明和安全提醒。

## 不应公网提供

- 新增或删除 WireGuard peer。
- 生成、下载或展示客户端私钥。
- 展示服务端私钥、完整服务器配置或真实管理端口。
- 无限制登录尝试。

## 如果未来必须公网管理

- 必须启用 HTTPS。
- 必须启用强密码和二次验证。
- 必须限制登录失败次数并记录审计日志。
- 后端使用最小权限执行管理命令。
- 管理 API 需要独立鉴权，不能只依赖前端隐藏按钮。
- 仍然优先考虑只允许可信来源或 VPN 内网访问。
